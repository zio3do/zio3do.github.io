# Ground Zero Portfolio

Personal website built with Astro. Phase 1 delivers a minimal static scaffold that runs locally and is ready for layout/content work.

## Commands

```sh
npm install
npm run dev
npm run build
npm run preview
```

## Project Structure

```
site-zero/
├── src/
│   ├── pages/         # Route-based pages
│   ├── components/    # Reusable components
│   ├── layouts/       # Page layouts
│   └── styles/        # Global and component styles
├── public/            # Static assets
└── astro.config.mjs   # Astro configuration
```
